# 용어 정리
## Network
- `SASE(Secure Access Service Edge, '사시' 또는 '새시'로 발음)` 는 네트워크 연결(SD-WAN)과 보안 기능(ZTNA, SWG, CASB 등)을 하나의 통합된 클라우드 서비스로 결합한 네트워크 보안 아키텍처입니다.
- `GRE(Generic Routing Encapsulation)`는 서로 다른 네트워크 사이에서 가상의 점대점(Point-to-Point) 연결을 생성하는 터널링 프로토콜

## 기타
- `Placement Matrix`란 "무엇(서비스/애플리케이션/리소스)을 어디(노드/서버/Account/리전 등)에 배치할지"를 표 형식으로 정리한 것
